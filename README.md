# Homebrew Tap

Homebrew tap for Andrew McCallum's command-line tools.

## Available formulae

| Formula | Description | Related repository |
|---|---|---|
| `hamta` | Run commands through a configurable proxy environment | [andrewmmc/hamta](https://github.com/andrewmmc/hamta) |

## Usage

Tap this repository:

```bash
brew tap andrewmmc/tap
```

Install `hamta`:

```bash
brew install hamta
```

Or install it directly without tapping first:

```bash
brew install andrewmmc/tap/hamta
```

Upgrade installed formulae from this tap:

```bash
brew update
brew upgrade hamta
```

Show formula information:

```bash
brew info hamta
```

Uninstall `hamta`:

```bash
brew uninstall hamta
```

Untap this repository:

```bash
brew untap andrewmmc/tap
```
